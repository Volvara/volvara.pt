O novo processo
5 Fases — clica para expandir
01
Prospecção — encontrar leads
30 min/semana · Segunda de manhã
semi-auto
▾
1
Instala o Agent-Reach (só uma vez)
O Agent-Reach dá ao Claude acesso ao Instagram, Maps, LinkedIn e mais — sem pagar APIs. Instala no teu Mac/PC com o terminal:
pip install https://github.com/Panniantong/agent-reach/archive/main.zip
agent-reach install --env=auto
Nota importante: Isto instala-se no teu computador — não aqui no Claude.ai. Abre o Terminal (Mac) ou CMD (Windows), cola o comando e prime Enter.
2
Pesquisa leads com 1 prompt
No Claude Code (no teu terminal), pede directamente:

O Claude devolve: tabela com 10-20 leads prontos para filtrar. Escolhes os 3 melhores.
3
Regista no Ops Dashboard
Copia os 3 melhores leads para o Ops Dashboard, coluna "Lead novo". Critérios: Instagram activo, 500+ followers, sector com template pronto.

Agent-Reach, Superpowers e spec-kit são ferramentas de estrutura — organizam o processo e automatizam a pesquisa. Os minions são os especialistas que operam dentro dessa estrutura. Dá sempre o contexto: briefing do cliente, o que já existe, e o output que queres.

Perguntas frequentes
Dúvidas comuns
Preciso de saber programar para usar estas ferramentas?▾
Não. O Agent-Reach instala-se com 2 comandos no terminal — copia e cola. O Superpowers instala-se com 1 comando no Claude Code. O spec-kit é 1 linha. Nenhuma destas ferramentas requer código. O Claude é o intermediário — tu dizes o que queres em português e ele executa.
O Agent-Reach pode ser banido do Instagram?▾
Existe risco com qualquer acesso por cookie. O Agent-Reach usa a tua sessão existente (como se fosses tu a navegar). Para minimizar: usa conta secundária dedicada para pesquisa, não faças centenas de pesquisas por hora, usa só para recolha de informação pública. Para 10-15 leads por semana o risco é muito baixo.
O que é o Claude Code e preciso dele?▾
Claude Code é o Claude que corre no terminal do teu Mac/PC. O Superpowers e o Agent-Reach funcionam melhor com Claude Code. Mas podes fazer tudo neste guia no Claude.ai normal — só perdes as funcionalidades automáticas do Superpowers. Para instalar: npm install -g @anthropic-ai/claude-code e depois claude no terminal.
Quanto tempo poupa o novo processo vs o actual?▾
Prospecção: de 1-2h manual para 20-30 min com Agent-Reach. Demo: de 3-4h do zero para 30-45 min com template. Publicação/SEO: de 1-2h manual para 30 min com checklists. Total: um ciclo completo passa de ~8h para ~2h. Isto significa 3-4 demos por semana em vez de 1.
Os templates já estão prontos ou tenho de os criar?▾
Já estão prontos. Tens 5 templates com todos os marcadores REPLACE_ identificados: restaurante, fotógrafo, bar/rooftop, serviços ao lar e clínica/saúde. Estão nos ficheiros volvara-template-*.html que já geraste. Carregas o ficheiro para o Claude e dás o briefing do cliente.
Tenho de fazer tudo isto de uma vez?▾
Não. O único passo urgente é instalar o Agent-Reach (5 min) para começares a prospecção automática esta semana. O Superpowers e spec-kit podes adicionar quando fizeres o próximo demo. O n8n e oh-my-claudecode só em Maio/Junho. Começa com o mínimo que dá impacto imediato: Agent-Reach + templates existentes.
