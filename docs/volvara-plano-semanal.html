FERRAMENTAS
AI

Avaliação honesta das 4 ferramentas — utilidade real para a Volvara, curva de aprendizagem, e recomendação clara de quando (e se) adoptar.

🧩
21ST.DEV
Marketplace de componentes React/UI — "npm para design engineers"
Volvara? Não agora
O que é
  • Biblioteca de componentes React com Tailwind e Radix UI
  • Instala com npx shadcn — código fica no teu projecto
  • Tem MCP server para usar com Cursor/Windsurf
  • 1.4M+ developers, backed by Y Combinator
Por que não agora
  • Stack Volvara = vanilla HTML/CSS — sem React, sem Tailwind
  • Componentes não são compatíveis com sites estáticos GitHub Pages
  • Adiciona complexidade sem benefício para o teu modelo actual
  • MCP pago — créditos esgotam rapidamente segundo reviews
Quando fica relevante: Quando a Volvara começar a construir em Framer (para clientes Presença) ou quando um cliente pedir uma web app/dashboard. Para sites estáticos simples, não traz valor.
🎙️
WISPR FLOW
Ditado por voz com IA — 4× mais rápido que escrever
Volvara? Sim — experimenta
O que faz
  • Converte voz em texto formatado em qualquer app (Gmail, Slack, Claude, VS Code)
  • Remove palavras de hesitação, formata automaticamente
  • 4× mais rápido que escrever — comprovado em reviews reais
  • Mac, Windows, iOS, Android — hotkey global
Limitações
  • 800MB RAM mesmo em idle — pesado
  • Voz processada na cloud — não para dados sensíveis
  • Plano gratuito: 2.000 palavras/semana — pode ser pouco
  • Startup 8–10 segundos
Uso concreto na Volvara: Ditar captions do Instagram, responder a emails de clientes, dar instruções ao Claude (este chat) por voz. Especialmente útil quando estás a trabalhar com o telemóvel ou a pensar em voz alta. Testa o plano gratuito esta semana nos briefings de clientes.
⚙️
N8N
Automação de workflows com IA — alternativa open-source ao Zapier
Volvara? Futuro próximo
O que faz
  • Liga apps entre si com lógica visual (drag-and-drop)
  • Integra Claude, OpenAI, Google Sheets, Gmail, Slack, etc.
  • Open-source — self-host grátis, cloud €20/mês
  • 500+ integrações, JS/Python nativo nos nodes
Para a Volvara
  • Automatizar: lead entra no formulário → Ops Dashboard → email notificação
  • Automatizar: cliente paga → email de boas-vindas → tarefa no CRM
  • Automatizar: post agendado Instagram com caption gerada por Claude
  • Curva de aprendizagem de ~2–3h para casos simples
Quando adoptar: Quando tiveres 3+ clientes pagantes e a gestão manual começar a consumir tempo. Por agora o Ops Dashboard cobre o que precisas. Marca para explorar em Maio — quando o portfólio estiver construído e o foco mudar para escalar.
📓
GOOGLE NOTEBOOKLM
Assistente de pesquisa — analisa os teus documentos com IA
Volvara? Sim — usa já
O que faz
  • Carregas PDFs, links, docs, YouTube — faz perguntas sobre eles
  • Respostas sempre baseadas nas fontes que carregaste (com citações)
  • Gera podcasts audio, summaries, guias de estudo, infográficos

Acção esta semana: Vai a notebooklm.google → cria um notebook "Volvara Ops" → carrega o volvara-memory.md e o website-playbook.html. A partir daí, quando precisares de consultar regras rápidas, perguntas lá em vez de procurar no ficheiro.

Como abordar amigos
A mensagem certa
Copy pronto
"Ei [nome], estou a lançar a minha agência de websites e precisava de mais uns projectos no portfólio. Vi que tens o [negócio] — achas que farias sentido ter um site? Critérios para escolher quem abordar
Perfil ideal para portfólio gratuito
Prioridade
Negócio real com clientes reais (não só projecto pessoal) · Sector diferente do que já tens (Mariana = babysitting) · Tem Instagram ou Facebook activo = fotos disponíveis · Mora perto ou conheces pessoalmente = briefing fácil · Vai recomendar activamente quando o site ficar bem
Sectores com mais potencial no Algarve
🍽️ Restauração
Cafés, restaurantes, pastelarias — quase todos têm mau site ou nenhum. Público local + turistas.
💆 Saúde & Bem-estar
Fisioterapeutas, psicólogos, massagistas — trabalham por marcação, formulário é essencial.
🔧 Serviços ao lar
Electricistas, canalizadores, pintores — quase zero presença online, enorme procura local.
🐾 Animais
Pet shops, veterinários, grooming — donos de animais são muito fiéis ao negócio que encontraram online.
Regras do portfólio gratuito
  • Máximo 3 sites gratuitos — a partir do 4º, cobras mesmo que seja amigo
  • Contrato simples: site grátis em troca de testemunho escrito + não pedes o site de volta
  • Defines o prazo — não te deixas arrastar por clientes que não entregam assets
  • Screenshots e URL vão directamente para o Instagram da Volvara — sem pedir permissão extra
  • Registo no Ops Dashboard como cliente (mensalidade €0 durante X meses, depois proposta de manutenção)
💡
O momento certo para cobrar: Quando tiveres 4–5 sites no portfólio e conseguires mostrar um case study concreto ("a Mariana passou de Instagram para site com SEO local em 3 dias"), a proposta de €299 setup + €49/mês deixa de precisar de justificação. O portfólio é a prova.
