VOLVARA OPS · SETUP

Configuração inicial — corre uma vez, apaga depois
1

Criar um GitHub PAT

Vai a github.com/settings/tokens/new.
Nome: volvara-ops · Expiration: 1 year
Scope necessário: repo → contents:write (só no repo volvara.pt)
Copia o token gerado — começa com ghp_

2

Criar os ficheiros JSON no repo

No repo volvara/volvara.pt, cria a pasta ops/data/ com estes 4 ficheiros:
leads.json clientes.json tarefas.json contab.json
Conteúdo de cada um: []

3

Encriptar o PAT com a tua password

Preenche abaixo. O PAT encriptado é guardado em localStorage — nunca sai do browser.

✓ Configuração guardada
O PAT foi encriptado com AES-256-GCM e guardado em localStorage.
Nunca é enviado para lado nenhum.

Próximos passos:
1. Apaga este ficheiro setup-pat.html do repo
2. Abre /ops/ e faz login com a tua password
3. Se mudares o PAT no futuro, volta a correr este setup
⚠ Abre este ficheiro localmente (não no servidor).
Depois de configurar, apaga-o do repo.
