VOLVARA
Ops Dashboard · Configuração Google Sheets
1

Criar a Google Sheet

Abre sheets.google.com e cria uma folha nova com o nome Volvara Ops. Copia o ID da folha — é a parte do URL entre /d/ e /edit.

2

Criar a API Key do Google

Vai a console.cloud.google.com → Create Credentials → API Key. Activa a Google Sheets API no projecto. Restringe a key ao domínio volvara.pt para segurança.

3

Tornar a Sheet pública para leitura

Na Sheet → Partilhar → Qualquer pessoa com o link pode ver. A escrita é feita via API Key autenticada.
